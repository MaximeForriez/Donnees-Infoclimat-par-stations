# Contenu des données

- [https://www.data.gouv.fr/organizations/association-infoclimat](https://www.data.gouv.fr/organizations/association-infoclimat)

- Métadonnées : [https://www.infoclimat.fr/stations/metadonnees.php?id=000RA](https://www.infoclimat.fr/stations/metadonnees.php?id=000RA)

Le dépôt contient toutes les données du site [infoclimat](https://www.infoclimat.fr/opendata/). Il s'organise par stations (nom d'un dossier), puis par année.

- [Codes pour télécharger les données de l'A.P.I.](./Telechargements-Codes/)

- [Données infoclimat](./infoclimat/)

> [!WARNING]
> Les données couvrent une période entre 1996 et le **15 novembre 2025**.

- [Liste des stations](./Telechargements-Codes/liste-des-stations.csv)

- [Métadonnées](./Telechargements-Codes/metadata.csv)

Chaque fichier en format C.S.V. contient :

- station_id

- dh_utc

- temperature

- pression

- pression_variation_3h

- humidite

- point_de_rosee

- visibilite

- vent_moyen

- vent_rafales

- vent_rafales_10min

- vent_direction

- temperature_min

- temperature_max

- pluie_1h

- pluie_3h

- pluie_6h

- pluie_12h

- pluie_24h

- pluie_cumul_0h

- pluie_intensite

- pluie_intensite_max_1h

- uv

- uv_index,ensoleillement

- temperature_sol

- temps_omm

- complements

- source
